🤖 HR Interview System Using Python & AI

An AI-powered HR Interview System built using Python, Streamlit, and advanced AI models like Gemini, GPT, and LLaMA.
This project automates the interview process by conducting Aptitude, Technical, and HR rounds with intelligent question generation and evaluation.

🚀 Features

📄 Resume Upload & Processing

🧠 AI-Generated Interview Questions

📝 Aptitude Round

💻 Technical Round

👨‍💼 HR Round

✅ Automatic Evaluation

📊 Instant Result Generation

🌐 Streamlit Web Interface

☁️ Google Cloud Storage Support

⚙️ Installation

Clone the repository:
git clone https://github.com/your-username/hr-interview-system.git

Move into the project folder:
cd hr-interview-system

Install dependencies:
pip install -r requirements.txt

Run the application:
streamlit run app.py
